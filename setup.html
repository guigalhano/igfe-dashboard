Como criar o GitHub Personal Access Token
É um passe que permite a esta página escrever ficheiros no teu repositório. Demora 2 minutos.
1
Abre as definições de tokens do GitHub
Vai a github.com/settings/tokens/new (ou GitHub → Foto de perfil → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token)
2
Preenche o formulário assim:

Scopes: Marca apenas repo

Como funciona o sistema
Tudo corre no teu navegador — nenhum servidor intermediário.
1
Tu → PDF do IGFE
Descarregas o PDF da Contraloría General quando publicam o relatório trimestral.
2
Claude AI extrai os dados
O PDF é enviado diretamente para a API da Anthropic (claude-sonnet-4-6). Claude lê todas as tabelas e gráficos e devolve um JSON estruturado com todos os valores numéricos.
3
Dados injetados no dashboard
O index.html atual é descarregado do GitHub, os novos dados substituem os antigos em todos os gráficos e KPIs, e o ficheiro atualizado é carregado de volta com um commit automático.
4
GitHub Pages publica em 1–2 min
O dashboard em garrinchashirts-create.github.io/igfe-dashboard/ fica atualizado automaticamente.
Segurança
• O token GitHub e a chave Anthropic são guardados apenas no localStorage do teu navegador.
• Nunca passam por servidores de terceiros — vão diretamente para api.github.com e api.anthropic.com.
• Podes limpar a qualquer momento no separador "Atualizar Trimestre".
